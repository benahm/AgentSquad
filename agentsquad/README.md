# agentsquad-cli

Projet CLI `agentsquad` initialise avec [commander](https://github.com/tj/commander.js).

## Installation

```bash
bun install
```

## Utilisation

```bash
bun run start --help
bun run ./bin/agentsquad.js hello --name Ahmed
```

Pour exposer la commande `agentsquad` globalement pendant le developpement :

```bash
bun link
agentsquad hello
```
